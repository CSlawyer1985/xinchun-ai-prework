## 📖 工具介绍

### 什么是Cherry Studio？

**Cherry Studio** 是一个**开源的可视化AI客户端**，它可以：

- 💬 多轮对话（支持多个AI模型）
- 📚 构建知识库（上传PDF、Word、Excel等文档）
- 🔧 构建助手（可复用的AI能力）
- 🎨 AI绘图（支持多种绘图模型）
- 🔌 支持MCP（Model Context Protocol）扩展

### Cherry Studio vs Claude Code

| 特性                | Claude Code      | Cherry Studio      |
| ------------------- | ---------------- | ------------------ |
| **界面**      | 命令行           | 可视化界面         |
| **上手难度**  | 需要熟悉命令行   | 点击操作，简单直观 |
| **知识库**    | 不支持           | 支持本地知识库     |
| **Skill开发** | 需要手动配置     | 可视化配置         |
| **适用场景**  | 批量处理、自动化 | 日常对话、知识管理 |

### 本课程的配置方案

本指南统一使用 **智谱GLM API**：

- 对话模型：`glm-4.7`（或平台最新可用GLM）
- 知识库嵌入模型：按平台可用配置（建议中文向量模型）
- 目标：先跑通单一稳定链路，再逐步扩展

> 说明：如果你已有其他可用模型 API，也可以在跑通 GLM 后自行接入作为补充。

计费建议（GLM）

- 先小额充值验证可用性
- 价格与模型可用性以智谱平台实时公示为准

## 📥 步骤1：下载Cherry Studio

### 1.1 访问官网下载

**操作说明**：

1. 打开浏览器，访问Cherry Studio官网：

   ```
   https://www.cherry-ai.com/
   ```
2. 首页右上角有"下载"按钮
3. 点击"下载"或访问直接下载页：

   ```
   https://www.cherry-ai.com/download
   ```
4. 下载页面显示多个平台的下载选项
5. 根据你的操作系统选择对应的版本：

   - **Windows**：点击"Windows Download"（下载 `.exe`文件，约150MB）
   - **macOS**：点击"macOS Download"（下载 `.dmg`文件，约180MB）
   - **Linux**：点击"Linux Download"（下载 `.AppImage`或 `.deb`文件，约150MB）

**✅ 成功标志**：

- 浏览器开始下载安装包
- 文件大小约150-200MB

**❌ 失败表现**：

- 下载速度极慢或卡住
- 提示"下载失败"

**🔧 解决方法**：

- **方法1**：检查网络连接
- **方法2**：使用GitHub下载链接（备用）：
  ```
  https://github.com/CherryHQ/cherry-studio/releases
  ```

**💡 小贴士**：下载时间取决于网速，通常需要2-10分钟。

### 💿 步骤2：安装Cherry Studio

### 2.1 Windows系统安装

**操作说明**：

1. **双击**下载好的 `.exe`安装包
2. 出现安装向导
3. 选择安装语言（中文/English）
4. 点击"下一步"
5. 选择安装位置
   - 默认：`C:\Users\YourName\AppData\Local\Programs\cherry-studio`
   - 建议使用默认位置
   - 点击"下一步"
6. 选择开始菜单文件夹
   - 默认：Cherry Studio
   - 点击"下一步"
7. 准备安装
   - 点击"安装"
8. 安装进度条
   - 等待安装完成（约1-2分钟）
9. 安装完成
   - 勾选"立即运行Cherry Studio"
   - 点击"完成"

**✅ 成功标志**：

- 安装向导显示"安装完成"
- Cherry Studio自动启动

**❌ 失败表现**：

- 安装过程中报错
- 提示"安装失败"

**🔧 解决方法**：

- **方法1**：以管理员身份运行安装包
- **方法2**：临时关闭杀毒软件
- **方法3**：检查磁盘空间是否充足（至少500MB）

---

### 2.2 macOS系统安装

**操作说明**：

1. **双击**下载好的 `.dmg`文件
2. 打开磁盘镜像
3. **将Cherry Studio拖拽到"应用程序"文件夹**
   - 拖拽图标
4. 等待复制完成（约30秒）
5. **启动Cherry Studio**：
   - 从"启动台"或"应用程序"文件夹中打开
6. 首次启动可能提示"无法打开"
   - 这是因为macOS的安全机制
   - **解决方法**：
     - 方法1：在"系统偏好设置 → 安全性与隐私"中点击"仍要打开"
     - 方法2：右键点击Cherry Studio → 打开 → 点击"打开"

**✅ 成功标志**：

- Cherry Studio出现在"应用程序"文件夹中
- 能正常启动

**❌ 失败表现**：

- 提示"已损坏"
- 提示"来自身份不明的开发者"

**🔧 解决方法**：

- **方法1**：在终端中执行（解除隔离）：
  ```bash
  xattr -cr /Applications/Cherry\ Studio.app
  ```
- **方法2**：在系统偏好设置中允许来自"App Store和被认可的开发者"的应用

---

### 2.3 Linux系统安装

**操作说明**：

#### 方法1：使用AppImage（推荐）

```bash
# 下载.AppImage文件（如果还没下载）
wget https://github.com/CherryHQ/cherry-studio/releases/download/v1.7.15/cherry-studio-1.7.15-linux-x86_64.AppImage

# 添加执行权限
chmod +x cherry-studio-*.AppImage

# 运行
./cherry-studio-*.AppImage
```

#### 方法2：使用.deb包（Ubuntu/Debian）

```bash
# 安装.deb包
sudo dpkg -i cherry-studio_1.7.15_amd64.deb

# 如果有依赖问题，执行：
sudo apt-get install -f
```

**✅ 成功标志**：

- Cherry Studio窗口打开
- 看到主界面

## 🎨 步骤3：首次启动与界面介绍

### 3.1 首次启动

**操作说明**：

1. **启动Cherry Studio**

   - Windows：从开始菜单或桌面快捷方式启动
   - macOS：从启动台或应用程序文件夹启动
   - Linux：从应用菜单或终端启动
2. 欢迎界面

   - 选择界面语言（推荐选择"简体中文"）
   - 选择主题（亮色/暗色/跟随系统）
3. 点击"开始使用"

**✅ 成功标志**：

- 看到Cherry Studio主界面
- 界面语言已设置为中文

---

### 3.2 界面介绍

Cherry Studio主界面分为4个区域

1. **左侧边栏**（最左侧）

   - 对话列表
   - 知识库入口
   - 设置按钮
2. **对话区域**（中间）

   - 显示对话历史
   - AI的回复和你的消息
3. **输入区域**（底部）

   - 文本输入框
   - 发送按钮
   - 附件按钮（上传文件）
4. **右侧边栏**（最右侧，可折叠）

   - 模型选择
   - 参数设置（温度、最大token数等）
   - Skill管理

**💡 小贴士**：如果你使用过网页版AI工具，Cherry Studio的界面会很熟悉。

## ⚙️ 步骤4：配置模型API（国内优先）

### 4.1 打开设置

**操作说明**：

1. 点击**左下角的设置图标**（齿轮图标）
2. 打开设置窗口
3. 在左侧菜单中选择**"模型服务"**或**"提供商"**

---

### 4.2 添加国内模型提供商（以GLM示例）

**操作说明**：

1. 模型服务设置页面

   - 显示已配置的AI提供商列表
2. 点击**"添加提供商"**或**"+"**按钮
3. 在提供商列表中选择**"智谱开放平台（Zhipu AI）"**

---

### 4.3 配置API Key

**操作说明**：

1. 智谱AI配置页面
2. 填写以下信息：

   - **API Key**：粘贴你在本指南中获取的API Key
   - **API地址**：`https://open.bigmodel.cn/api/paas/v4/`
   - **模型名称**：`glm-4.7`（或你选择的智谱模型）
3. 填写示例

   ```
   API Key: sk-abc123.xyz789.abcdef123456
   API地址: https://open.bigmodel.cn/api/paas/v4/
   模型名称: glm-4.7
   ```
4. 点击**"保存"**或**"测试连接"**

**✅ 成功标志**：

- 显示"连接成功"或"测试通过"
- 智谱AI出现在提供商列表中

**❌ 失败表现**：

- 显示"连接失败"
- 显示"401 Unauthorized"
- 显示"API Key无效"

**🔧 解决方法**：

- **方法1**：检查API Key是否正确复制（注意不要有多余空格）
- **方法2**：检查API地址是否正确：`https://open.bigmodel.cn/api/paas/v4/`
- **方法3**：登录智谱AI平台检查API Key是否有效
- **方法4**：检查账户余额是否充足

---

### 4.4 添加模型到列表

**操作说明**：

1. 在模型设置中，找到**"管理模型"**或**"模型列表"**
2. 点击**"添加模型"**
3. 添加模型对话框
   - 选择提供商：智谱开放平台
   - 选择模型：**GLM-4.7**（或平台当前可用的GLM模型）
   - 点击"添加"

**✅ 成功标志**：

- GLM-4.7出现在模型列表中
- 可以在对话界面选择该模型

---

**📍 检查点**：在继续之前，请确认：

- [ ] 智谱GLM已成功添加到提供商列表
- [ ] API Key配置正确
- [ ] 连接测试通过
- [ ] GLM-4.7模型已添加到模型列表

---

## 💬 步骤5：创建第一个对话

### 5.1 新建对话

**操作说明**：

1. 点击**左上角的"助手"**按钮并添加助手***
2. 新建对话界面
3. 在右侧边栏的**顶部**下拉菜单中，选择**"GLM-4.7"**

---

### 5.2 发送测试消息

**操作说明**：

1. 在底部的输入框中输入：

   ```
   你好，请介绍一下你自己。
   ```
2. 点击**发送按钮**（或按 `Enter`键）
3. 等待AI响应（通常1-3秒）

**预期输出**：

```
Assistant: 你好！我是智谱AI开发的大语言模型GLM-4。我可以帮助您回答问题、提供信息、进行对话交流等。请问有什么可以帮助您的吗？
```

**✅ 成功标志**：

- 消息发送成功
- 收到AI的回复
- 回复内容正常（中文、流畅）

**❌ 失败表现**：

- 显示"发送失败"
- 显示"网络错误"
- 长时间无响应

**🔧 解决方法**：

- **方法1**：检查网络连接
- **方法2**：检查API Key是否正确配置
- **方法3**：检查模型是否正确选择
- **方法4**：重新启动Cherry Studio

---

### 5.3 多轮对话测试

**操作说明**：

1. 继续在输入框中输入：

   ```
   我是一名律师，你能帮我做什么？
   ```
2. 发送消息

**预期输出**：

```
Assistant: 作为一名律师，我可以在以下方面帮助您：

1. 法律检索：帮您查找相关法律法规、司法解释和案例
2. 文书起草：协助起草合同、起诉状、答辩状等法律文书
3. 案件分析：分析案件事实、识别争议焦点、提供法律意见
4. 风险评估：评估合同条款的法律风险
5. 知识管理：帮您构建法律知识库，快速检索所需信息

您希望我从哪个方面开始帮助您？
```

**✅ 成功标志**：

- 能进行多轮对话
- AI能理解上下文
- 回复内容专业且相关

## 📎 步骤6：上传文档测试

### 6.1 上传文档

**操作说明**：

1. 准备一个测试文档（PDF、Word、TXT等）
2. 点击输入框上方的**附件图标**（📎）
3. 文件选择对话框
4. 选择测试文档并打开
5. 文档出现在输入框上方
6. 输入提示词：
   ```
   请总结这份文档的主要内容。
   ```
7. 点击发送

**预期输出**：

```
Assistant: 这份文档的主要内容是...
[AI总结的文档内容]
```

**✅ 成功标志**：

- 文档上传成功
- AI能读取文档内容
- 总结内容准确

**❌ 失败表现**：

- 提示"文件格式不支持"
- 提示"文件过大"
- AI无法读取文档内容

**🔧 解决方法**：

- **方法1**：检查文件格式（支持PDF、Word、TXT、Markdown等）
- **方法2**：检查文件大小（建议<10MB）
- **方法3**：尝试使用其他文档测试
