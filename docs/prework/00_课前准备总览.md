## 📋 课程机制（重要）

### 1. 公益课堂 + 保证金约束

- 本课程本质上是公益课堂，保证金不是学费
- 设保证金是为了帮助大家形成学习约束：认真听课、认真做作业、积极互动
- 退还基础：完成听课与作业要求（好坏不作为唯一标准，完成最重要）

### 2. 课程节奏（固定）

- 原则上每周一堂，安排在**周日晚上**
- 每次上课后给你**一整周**做练习和作业，用于消化吸收
- 核心课共**5次**，后续会视情况邀请技术嘉宾加餐分享（可选）

### 3. 退出与退还机制

- 如你在学习中觉得听不懂或内容过于简单，可中途退课
- 中途退课时按约定扣除对应额度后，退还剩余保证金
- 具体细则以报名说明/学习约定为准

### 4. 互动共学机制

- 我们鼓励分享真实法律场景中的问题和作业
- 课程目标之一是寻找同频法律人，形成长期共学网络

## 🔑 账号准备

### 必须准备的账号（至少2类）

**1. 国内模型账号（至少1个）**

- 候选：智谱GLM / Kimi / DeepSeek
- 用途：保证中国大陆网络环境下可稳定直连
- 建议：先开通一个主力账号，充值小额测试预算

**2. 国际模型账号（至少1个）**

- 候选：OpenAI（ChatGPT）/ Anthropic（Claude）/ Google（Gemini）
- 用途：对比能力边界，覆盖高质量写作和复杂推理场景
- 注意：多数情况下需要VPN或企业合规出口网络

**✅ 建议最低组合：1个国内可直连 + 1个国际模型备用。**

### 计费与接入限制（务必先看）

- 高频使用（连续写作、长时协作）优先评估 `Coding Plan`（订阅）
- 低频或波动任务优先 `PAYG`（按量计费），并设置月度预算上限
- 不同平台存在调用速率、并发、模型白名单等限制，开课前先完成一次真实调用测试
- 国际平台可能有地区/手机号/支付方式限制，请预留注册和验证时间

### 主流模型速览（2026年2月口径）

| 地区 | 模型家族 | 最新口径（按平台可用） | 强项                   | 注意点                   | 网络建议           |
| ---- | -------- | ---------------------- | ---------------------- | ------------------------ | ------------------ |
| 美国 | ChatGPT  | GPT-5 / GPT-5.1        | 通用推理、策略草拟     | 成本偏高                 | 多数需VPN/合规出口 |

## 🆘 获取帮助的方式

### 第一层：查阅文档

- **问题类型**：安装失败、配置错误、命令不识别等
- **解决方法**：查阅《04_常见问题FAQ.md》
- **响应时间**：即时

### 第二层：微信群求助

- **问题类型**：文档中没有覆盖的问题、需要人工判断的问题
- **解决方法**：在学员微信群中提问
- **响应时间**：通常1-2小时内
- **提问技巧**：
  - 描述你在做什么步骤
  - 截图显示错误信息
  - 说明你的操作系统（Windows/Mac/Linux）

### 第三层：截图求助AI（终局建议方案，也是你应该养成的习惯）

- **问题类型**：文档和微信群都无法解决的问题
- **解决方法**：将错误截图发给网页版AI，寻求解决方案
- **推荐工具**：
  - 网页版Claude：https://claude.ai/
  - 网页版ChatGPT：https://chat.openai.com/
  - KIMI：https://www.kimi.com/
- **提示词模板**：
  ```
  我在安装[工具名称]时遇到了错误，请帮我分析原因并提供解决方案。

  我的操作系统：[Windows 10/Mac macOS 13/Ubuntu 20.04]
  我在执行的步骤：[简述你正在做什么]
  错误信息：[粘贴错误信息或截图描述]

  请提供详细的解决步骤，谢谢！
  ```

**⚠️ 注意**：

- 截图时请包含完整的错误信息（不要只截一半），如果是命令行代码，可以直接复制代码内容提供
- 不要泄露敏感信息（如API Key、密码等）
- 提问时请尽量详细，这样AI才能更准确地帮助你
